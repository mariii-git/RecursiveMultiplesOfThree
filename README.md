Recursive Multiples of Three Finder (C#)

A C# program that finds all multiples of 3 less than a given input number using a recursive method.

Features

· Recursive implementation to find multiples of 3
· Processes user input from console
· Outputs all qualifying numbers
· Demonstrates recursion with conditional logic

Technologies

· C# (.NET Console Application)

Usage

1. Run the program
2. Enter an integer when prompted
3. View all multiples of 3 smaller than the entered number

csharp
// Example output:
// Enter a number: 20
// Multiples of 3 less than 20:
